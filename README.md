
# Laundry Wallah Website

## Overview

Laundry Wallah is a simple and responsive laundry service website created using pure HTML. The website showcases the services offered by a laundry business, pricing information, image gallery, and an appointment booking form for customers.

## Features

* Attractive landing page with company logo and welcome message
* Laundry service description section
* Image gallery displaying laundry operations and finished products
* List of services offered
* Pricing table for different laundry services
* Benefits of choosing Laundry Wallah
* Customer appointment booking form
* Footer with company information, contact details, and quick navigation links
* Internal page navigation using anchor links

## Technologies Used

* HTML5
* Basic Inline CSS Styling
* Images for branding and service showcase

## Website Sections

Displays the company logo, tagline, and welcome message.

Shows images related to laundry machines, ironing services, and finished products.

### Footer

## How to Run

1. Download or clone the repository.
2. Ensure all images are placed inside the `Images` folder.
3. Open `index.html` in any modern web browser.
4. Explore the website and test the appointment booking form.
